Color System

Primary Yellow는 “감정(동기/환영)”을 담당합니다. 정보는 Blue, 성취는 Green, 텍스트는 Ink, 배경은 Warm Background가 담당합니다.

Usage Rules

상황 권장 색 금지
주요 행동(CTA) Primary Yellow 빨강을 기본 강조로 사용
링크/힌트/정보 Learning Blue 노랑으로 링크 표현
정답/완료 Success Green 정답을 “색만”으로 전달
본문 텍스트 Ink Black 노랑 배경 위 흰 글자
배경 Warm Background 완전 흰 배경만 고집

Anti-patterns (Colors)

  • 노랑 배경 + 흰 글자 (가독성 급락)
  • 버튼마다 다른 색 (학습 UI에서 혼란 유발)
  • 의미 없는 색 추가 (유지보수 지옥)
  • 빨강을 기본 강조 (불안/시험 톤 강화)

자세한 금지 목록은 Anti-patterns 페이지를 참고하세요.
