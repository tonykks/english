Typography

Pretendard는 한/영 혼용 학습 UI에서 균형이 좋습니다. 학습용 문서는 “과한 장식”보다 “일관된 위계”가 핵심입니다.

Font

Pretendard (Web CDN)

사용 이유: 가독성, 한/영 균형, 웹 최적화, 무료/안정적 배포.

CSS Snippet

styles.css에 이미 반영되어 있습니다.


font-family: "Pretendard", -apple-system, BlinkMacSystemFont, system-ui, sans-serif;
line-height: 1.6;

Anti-patterns (Typography)

  • 굵기 3종 이상 난립 → 위계가 아니라 소음이 됨
  • 줄간격 1.3 이하 → 학습 UI에서 피로 증가
  • 영어만 다른 글꼴 → 시각적 분열감 발생
  • 전체 대문자 남용 → 읽기 속도 저하
